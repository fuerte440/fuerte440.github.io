Generador de informes

Guarda tus facturas y genera tus informes.

¿Qué es esta aplicación?

Es una aplicación muy sencilla en la cual se generan facturas que se graban en la base de datos con un DNI ya existente en ella, para despues generar un informe con esas facturas grabadas con ese DNI.

Capturas de pantalla

Funciones principales

  • 📂 Grabar facturas en la base de datos.
  • 📊 Exportar el informe.
  • 🔒 Validación del DNI para generar las facturas.

Tecnologías utilizadas

  • NetBeans (Java).
  • SQLite.
  • Jaspersoft.
