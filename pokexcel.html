Pokexcel

Tu colección Pokémon organizada, digital y exportable.

¿Qué es Pokexcel?

Pokexcel es una aplicación Android desarrollada con Kotlin y pensada para coleccionistas de cartas Pokémon. Permite gestionar y digitalizar colecciones de forma intuitiva, añadiendo cartas manualmente o escaneándolas con la cámara del dispositivo. El usuario puede crear listas personalizadas por expansión, marcar cartas repetidas y exportar sus listas a Excel para llevar un control ordenado o compartirlas fácilmente. La app incluye funcionalidades como inicio de sesión seguro, integración con bases de datos locales (SQLite), y en el futuro se integrará con APIs externas como eBay o Cardmarket para estimar el valor de las cartas. Pokexcel es ideal tanto para coleccionistas novatos como para expertos que quieran mantener su colección siempre actualizada desde el móvil.

Funciones principales

  • 🎴 Añadir cartas manualmente y por escaneo (función en desarrollo con margen de mejora).
  • 📂 Crear listas personalizadas basadas en colecciones oficiales.
  • 📊 Exportar listas a archivos Excel (.xlsx) para compartir o imprimir.
  • 🔒 Inicio de sesión con cuenta propia o mediante Google (autenticación segura).
  • 📷 Uso de la cámara para capturar el número de carta (detección básica implementada).
  • 📡 Integración futura con APIs externas como Cardmarket o eBay (no disponible aún).

Tecnologías utilizadas

  • Android Studio (Java y Kotlin).
  • SQLite + Apache POI.
  • SoundPool y Camera API.
  • Firebase Authentication (futura integración).
